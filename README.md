# 🎉 Cinematic Birthday Surprise

A beautifully designed **interactive birthday surprise website** built using **HTML, CSS, and JavaScript**.
This project creates a cinematic experience with animations, transitions, music, and a memory gallery.

---

## 🌟 Features

* 🎬 Cinematic intro (Hero section)
* 💌 Interactive message slider
* 💡 Celebration effects (lights, music, balloons)
* 🎭 Curtain opening animation (surprise reveal)
* 🖼️ Memory gallery with hover effects
* ❤️ Floating hearts & sparkle animations
* 🎵 Background birthday music
* 📱 Fully responsive design

---

## 🛠 Tech Stack

* HTML5
* CSS3 (Animations, Glassmorphism, Gradients)
* JavaScript (DOM, Animations, Effects)
